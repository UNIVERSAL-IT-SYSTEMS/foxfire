# foxfire

> Firefox Launcher with clean and custom profiles without WebDriver

### Install

Install node.js (npm should be available in your terminal app)

#### Module

```
npm i foxfire --save-dev
```

#### Global Command-line

(Install might require `sudo`)

```
npm i -g foxfire
```
